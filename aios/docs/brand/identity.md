# AI-OS Brand Identity

## Name

**AI-OS** (Artificial Intelligence Operating System)
Codename: **CC2** (Command Center 2)

## Operator

**Chris** — Sole authorized operator. Identity enforced via `identity.lock`.

## Visual Identity

### Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Cyan Primary | `#00E5FF` | Primary accent, borders, headings |
| Cyan Light | `#66FFFF` | Highlights, hover states |
| Background Dark | `#0A0A0A` | Main background |
| Background Mid | `#1A1A1A` | Panel backgrounds |
| White | `#FFFFFF` | Primary text |
| Green | `#00FF88` | Status OK indicators |
| Red | `#FF4444` | Error / fault indicators |
| Yellow | `#FFCC00` | Warnings |
| Dim | `#888888` | Secondary text |

### Typography

- **Font**: System monospace stack — `'Courier New', 'Lucida Console', 'DejaVu Sans Mono', monospace`
- **No external font loading** — all fonts are system-provided
- Terminal aesthetic: fixed-width, high contrast

## Design Principles

1. **Dark Cyberpunk**: Dark backgrounds, cyan/green accents, monospace text
2. **Information Dense**: Maximize useful data on screen
3. **Terminal First**: Designed for terminal and text interfaces
4. **No External Dependencies**: All assets self-contained, no CDNs
